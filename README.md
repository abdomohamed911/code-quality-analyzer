# Code Quality Analyzer

A static code analysis tool for JavaScript/TypeScript that evaluates code quality across multiple dimensions including complexity, naming conventions, error handling, and documentation coverage.

## Features

- **Cyclomatic Complexity Analysis**: Identifies functions with high complexity that may need refactoring
- **Naming Convention Checker**: Validates camelCase, PascalCase, and UPPER_SNAKE_CASE patterns
- **Error Handling Audit**: Checks for proper try-catch blocks in async functions
- **Documentation Coverage**: Measures JSDoc coverage for functions
